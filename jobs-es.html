Acerca de Sancrisoft

Sancrisoft es una empresa conformada por un grupo de profesionales altamente capacitados con una sólida experiencia en el desarrollo de soluciones basadas en Internet. Trabajamos con las tecnologías más modernas, a precios competitivos, con foco en la calidad pero con un fin que no perdemos de vista ¡el objetivo del negocio del cliente! Somos especialistas en la creación de sitios web usando ReactJS una potente librería para crear interfaces de usuario interactivas, nos apasiona crear aplicaciones móviles para iOS y Android así como también sistemas de información desarrollados a la medida.

Conoce más de nuestros servicios y las tecnologías con las que trabajamos.

Descripción del Puesto

Estamos buscando un Desarrollador Full Stack para unirse a nuestro equipo, trabajamos en múltiples proyectos para clientes distribuidos en diferentes paises. En este puesto aplicarás tus habilidades de HTML5 / CSS3 / JS y aprenderás de tecnologías web modernas como NodeJS, Express, ReactJS, Redux, Webpack, así como tambien React Native para construir Aplicaciones Móviles. Trabajarás en tareas de Front-End y de Back-End para construir productos de primera categoría, buscamos personas talentosa con buena actitud frente a la resolución de problemas esforzándose en la calidad de su trabajo.

Tenemos proyectos interesantes en los que puedes trabajar, si quieres ser parte de uno de ellos sigue leyendo y dejanos un mensaje.

Responsabilidades

  • Crear aplicaciones web elegantes, eficientes y mantenibles para respaldar y ampliar nuestros proyectos actuales
  • Solucionar problemas reportados por los clientes
  • Convertir diseños en templates HTML / CSS / JS
  • Colaborar con miembros del equipo y los clientes

Requerimientos

  • Dominio de sistema de control de versiones Git
  • Conocimiento avanzado de HTML5, CSS3 y Javascript
  • Conocimiento de problemas de compatibilidad entre navegadores y las formas de solucionarlo
  • Comprensión general de principios de diseño UI/UX y mejores prácticas
  • Comprensión general de los principios establecidos en Section 508
  • Inglés básico y buenas habilidades de comunicación
  • Autodidacta y altamente motivado
  • Proactivo y solucionador de problemas
  • Apasionado por el desarrollo web, empoderado y orgullo de su trabajo
  • Dispuesto a ayudar a sus compañeros, compartir su conocimiento y aprender de ellos
  • Abierto a recibir feedback

Habilidades Deseables

  • Node y npm
  • Conocimiento básico de ES6
  • ReactJS y React Native
  • Webpack
  • Redux
  • Foundation / Bootstrap
  • Experiencia construyendo aplicaciones iOS y Android
  • CMS such as Drupal
